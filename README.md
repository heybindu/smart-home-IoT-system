# Smart Home IoT System

## Overview

Developed a Smart Home IoT System during the 24-hour TCE Affinity 2025 Hackathon. The project focused on automating household devices using IoT technologies to improve convenience and energy efficiency.

## Features

* Home appliance automation
* IoT-based device control
* Sensor integration
* Real-time monitoring
* Smart home management

## Technologies Used

* Arduino
* IoT Concepts
* Sensors
* Embedded C

## Team Contribution

Collaborated with team members to design and implement an automated home control solution within the hackathon timeframe.

## Applications

* Smart Homes
* Energy Management
* Home Automation Systems
